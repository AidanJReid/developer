The Daily Rot

Welcome to the coding hub for The Daily Rot - a project where I create a new website every single day until I land my first job as a software developer.

Why?!

The long version

The Shorter Version - I firmly believe that daily, practical application is the best way to learn a new skill.

Throughout this project my mission is to improve my skills as a developer and secure a job in what is a highly competitive industry.

These sites will be influenced by ROT - a collection of dark humour short stories which I wrote - now reimagined and reanimated with code!

But enough yapping. Let's get coding.
