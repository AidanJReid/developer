ROT

Welcome to the ROT Project - an interactive hub where I reanimate and reimagine the short stories I've written and bring them to life through coding magic!

The ROT collection is a set of dark humour short stories I wrote back in 2017 and they are all free to download.

These mini-sites are all influenced by events in the ROT Universe which (currently) spans six books.

But enough yapping. Let's get coding.
